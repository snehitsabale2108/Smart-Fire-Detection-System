# Smart-Fire-Detection-System
The Smart Fire Alarm System is an ESP32-based mini project designed to detect fire and smoke using flame and gas sensors. The system provides instant alerts through a buzzer and LED indicators, helping improve safety and reduce fire-related risks in homes, offices, and other environments.

---

## Features
- Fire and smoke detection
- Real-time monitoring
- Buzzer alert system
- LED status indication
- Low-cost and easy to build

---

## Components Used
- Arduino UNO
- Flame Sensor
- MQ-2 Smoke Sensor
- Buzzer
- Red LED
- Green LED
- Resistors
- Breadboard
- Jumper Wires
- USB Cable

---

## Working
1. Sensors continuously monitor fire and smoke levels.
2. Arduino processes sensor data.
3. If fire/smoke is detected:
   -  Red LED turns ON
   -  Buzzer activates
4. Under normal conditions:
   -  Green LED remains ON
